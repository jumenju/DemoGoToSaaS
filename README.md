# DemoGoToSaaS

Site de démonstration **go TO jOHN** — showcase d'un projet SaaS hébergé sur GitHub Pages.

---

## Structure du projet

```
DemoGoToSaaS/
├── index.html   # Page principale : heading, tableau changelog, footer
└── README.md    # Documentation du projet
```

### Détail des fichiers

| Fichier | Rôle |
|---|---|
| `index.html` | Page unique du site : en-tête avec titre, section changelog, footer |
| `README.md` | Présentation du projet, structure et technologies |
